⚡ Ключевые особенности
🎨 Визуальная система
Primary color: #6366F1 (Indigo) · Accent: #10B981 (Emerald) · Urgent: #EF4444 (Red) · Font: Merriweather (headings) + Inter (body) · Border-radius: 16px · Animations: 200–600ms ease-out
🌙 Темная тема
Полная поддержка темной темы на всех экранах. Переключение темы сохраняется в localStorage. Dark background: #0F172A · Smooth transitions между темами.
📱 Адаптивность
Mobile-first подход. Все экраны работают на мобильных устройствах (нижняя навигация), планшетах и десктопах (полная версия). Используется Tailwind CSS и custom media queries.
✨ Интерактивность
Плавные переходы, hover-эффекты, animated counters, placeholder cycling, theme toggle, tab switching, smooth scrolling, validated forms.
🗺 Карта (Leaflet.js)
Интеграция Leaflet.js на главной странице. Динамическое добавление маркеров, wave animation, zoom-in при загрузке, поддержка темной темы.
🚀 Production-ready
Все экраны могут быть развернуты как стоит-alone HTML файлы или интегрированы в React/Vue приложение. Код хорошо организован и документирован.
