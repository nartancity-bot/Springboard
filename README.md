# 🚀 Трамплин — Прототип платформы

Полнофункциональный прототип платформы для поиска работы, стажировок, ивентов и нетворкинга.

---

## 📁 Файлы проекта

### 1. **index.html** — Главная страница навигации
Стартовая точка. Содержит ссылки на все созданные экраны, статистику проекта и описание ключевых особенностей.

**Открыть:** `index.html`

## 🎯 Публичные экраны (гость)

### 2. **trampoline-home.html** — Домашняя страница
Главная страница с тремя режимами просмотра.

**Компоненты:**
- 🎨 Хедер (фиксированный 56px) с логотипом, кнопками входа/регистрации, темой
- 🗺 Hero-момент с картой Leaflet.js (zoom-in анимация 1.2s)
- 📊 Счётчик возможностей (анимирует от 0 до 312)
- 🔍 Поисковая строка с чередующимся placeholder
- 🏷 Горизонтальная полоса фильтр-чипсов
- 3️⃣ Три режима: **Карта** (Leaflet) | **Лента** (вертикальный скролл) | **Свайп** (стек карточек)
- 💳 Карточки возможностей с hover-эффектами
- 📱 Нижняя навигация (мобиле)

**Интерактивность:**
- Mode switching между карта/лента/свайп
- Theme toggle (темная тема)
- Animated counter
- Placeholder cycling каждые 3 секунды
- Leaflet.js карта с flyTo zoom
- Staggered card animations

### 3. **trampoline-opportunity-card.html** — Карточка возможности
Подробная страница о конкретной вакансии/стажировке.

**Компоненты:**
- 📸 Медиа-обложка (градиент или фото офиса)

  - Место (мини-карта, адрес, как добраться)
- 📌 Фиксированный CTA-блок внизу (В избранное + Откликнуться)

**Интерактивность:**
- Tab switching
- Favorite toggle (синхронизация между кнопками)

- Theme toggle

## 🎨 Визуальная система

### Цветовая палитра
- **Primary:** `#6366F1` (Indigo) — основной цвет для UI
- **Accent/CTA:** `#10B981` (Emerald) — кнопки действий
- **Urgent:** `#EF4444` (Red) + pulse — срочные вакансии, ошибки
- **Gold:** `#F59E0B` — верификация, избранное
- **Bg Light:** `#F8FAFC` — светлый фон
- **Bg Dark:** `#0F172A` — темный фон

### Типография
- **Headings:** Merriweather (serif) — элегантная, заметная
- **Body:** Inter (sans-serif) — чистая, разборчивая
- **Letter-spacing:** -0.02em на заголовках
- **Font weights:** 400, 500, 600, 700

### Компоненты
- **Border-radius:** 16px для карточек, 8px для кнопок, 6px для чипсов
- **Shadows:** Многоуровневые, цветные shadows с низкой opacity
- **Animations:** 200–600ms ease-out для всех переходов
- **Line-height:** 1.6–1.7 для удобочитаемости

## ✨ Ключевые особенности

### Анимации
- 🎬 **Hero-момент** — zoom-in карты при загрузке
- 📊 **Counting animation** — счётчик от 0 до 312
- 💳 **Card animations** — staggered appearance
- 🌊 **Wave effect** — маркеры на карте появляются волной
- ✨ **Smooth transitions** — 200–600ms ease-out для всех переходов

### Интерактивность
- 🎯 **Immediate feedback** — hover/active состояния на всех элементах
- 📝 **Form validation** — реальная валидация с сообщениями об ошибках
- 🔄 **Stateful components** — кнопки меняют вид при клике
- 🔍 **Searchable inputs** — анимированный placeholder, focus states
- 🗺 **Interactive map** — Leaflet.js с zoom и clustering

### Performance
- ⚡ **Vanilla JS** — без тяжёлых зависимостей (только Leaflet для карты)
- 🎨 **Tailwind CSS** — CDN версия для быстрого прототипирования
- 📦 **Лёгкие файлы** — 16–25kb на экран
- 🚀 **Быстрая загрузка** — все анимации на CSS/JS, без изображений

## 🎨 Кастомизация

### Изменить основной цвет
Найдите в файле:
```css
--color-primary: #6366F1;
```
И замените на нужный.

### Изменить шрифты
В `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@400;700&display=swap" rel="stylesheet">
```

### Добавить анимацию
Используйте `animation` или `transition` в CSS:
```css
transition: all 0.2s ease;
animation: slideDown 0.6s ease;
```
